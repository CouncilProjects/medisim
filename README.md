Μια πλατφόρμα προσομοίωσης ΜΕΘ.

Φτιάχτηκε στα πλαίσια απαλλακτικής εργασίας για το ΠΑΔΑ.

* **Frontend:** [React](https://react.dev/)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Build Tool:** [Vite](https://vite.dev/)
